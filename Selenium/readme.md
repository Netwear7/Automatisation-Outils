# Selenium

## Informations

Selenium est un ensemble d’outils pour l’automatisation du navigateur Web. Il permet de contrôler à distance les instances du navigateur et émuler l’interaction d’un utilisateur avec le navigateur.

Différents outils :
* WebDriver
* IDE
* Grid
* Remote Control
